# RTFOAM v9

Final four-architecture release: Trainer, Fighter, Experimental flying wing, and Warbird.

Each generated aircraft is solved against a deterministic architecture and rendered in three distinct hangar views. The detail page contains a precise SVG build dossier and a separate 1:1 cut sheet.

Upload every file and the full `knowledge/` folder together. Configure `OPENAI_API_KEY`, `OPENAI_MODEL=gpt-5.6`, and `OPENAI_REASONING_EFFORT=high` in the Cloudflare Worker.
